## Objetivo
Diante das recentes oscilações no preço da gasolina, influenciadas por fatores geopolíticos e econômicos globais, este trabalho tem como objetivo analisar a evolução dos preços dos combustíveis no período de 1970 a 2026. A partir do dataset selecionado, busca-se identificar padrões, tendências e as variações nos preços ao longo do tempo.
